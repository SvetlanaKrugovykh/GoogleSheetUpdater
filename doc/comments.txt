Для извлечения ссылок на фото из Google Таблицы, где ссылки скрыты за текстом, можно использовать скрипт Google Apps Script, чтобы автоматически получить ссылки. Вот пошаговое руководство:
Шаг 1: Открытие редактора скриптов
Откройте таблицу в Google Таблицах.
В верхнем меню выберите Extensions (Дополнения) → Apps Script.
Шаг 2: Написание скрипта
В открывшемся редакторе скриптов вставьте следующий код: services\extractLinks.js
Нажмите File → Save и дайте скрипту имя, например Extract Links.
Шаг 3: Запуск скрипта
В редакторе скриптов выберите функцию extractLinks в выпадающем списке.
Нажмите значок ▶ (Run).
Примите запросы на авторизацию (если появятся).
Шаг 4: Результат
После выполнения скрипта:
Будет создан новый лист в вашей таблице с названием Extracted Links, содержащий все найденные ссылки.

2. Создайте и настройте Google API проект
Перейдите в Google Cloud Console.
Создайте проект и включите Google Sheets API и Google Drive API.

Ошибка "You need access" возникает, потому что новая таблица создается с правами доступа только для владельца (вашего сервисного аккаунта, а не вашего Google аккаунта). Чтобы вы могли открыть файл по ссылке, нужно вручную или программно предоставить доступ вашему Google аккаунту.

Решение: программное добавление доступа
Google Sheets API позволяет управлять правами доступа через Google Drive API. Нужно добавить ваш Google аккаунт (или других пользователей) в качестве редактора/чтеца.
